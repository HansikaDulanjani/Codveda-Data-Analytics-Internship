
# Codveda Data Analytics Internship

This repository contains my completed work for the Codveda Technologies Data Analytics Internship Project. The project covers the complete data analytics workflow, including data preprocessing, exploratory data analysis, visualization, machine learning, business intelligence, and natural language processing.

## Tasks Completed

1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Data Visualization
4. Regression Analysis
5. Time Series Analysis
6. K-Means Clustering
7. Customer Churn Prediction
8. Power BI Dashboard Development
9. NLP-Based Sentiment Analysis

* Cleaned and analyzed real-world datasets.
